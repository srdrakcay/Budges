
# Budges-Harcama Takip Uyulaması

You can track and change your Expenses in this application.You can also follow crypto prices instantly.

## Application Features
- Swipe to Delete
- Crypto Price
- Simple Notification
- Spending Summary


## Project Features

- MVVM Architecture 
- LiveData
- ViewModel
- ViewPager 2
- Room
- Retrofit
- Navigation Component
- Courotines
- MPAndroid Bar Chart
- Tab Layout
- Json Pars
- Repository
- NotificationManager
- Custom Drawable Button And Bottom Bar Style

## Design Features
- Dots (Viewpager Slider)
- Custom Drawable Button And Bottom Bar Style
- Figma
- Canva
